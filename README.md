# BangumiTVBot

A telegram bot (@BangumiTVBot) to provide ranking from bangumi.tv.

可以实现搜索动画和获得排行榜

排名信息来自 Bangumi.tv

## Usage

Packages needed:

- Axios
- cheerio
- node-telegram-bot-api

Create `token.js` and export the token and baseURL string. Then execute the following commands:

```shell
# npm start
```
